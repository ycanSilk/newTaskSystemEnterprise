# 平台服务须知

## 1. 服务概述

欢迎使用我们的平台服务。本平台服务须知（以下简称"须知"）旨在向您说明平台的服务内容、使用规则、收费标准等重要信息，帮助您更好地了解和使用平台服务。

## 2. 服务内容

### 2.1 任务发布与管理
- 发布各种类型的任务，包括但不限于抖音评论任务、点赞任务等
- 管理任务状态，包括待审核、进行中、已完成等
- 查看任务详情和统计数据

### 2.2 账号租赁服务
- 发布账号出租信息
- 浏览和申请账号租赁
- 管理租赁订单和交易

### 2.3 支付与钱包管理
- 充值和提现功能
- 查看交易记录和余额
- 设置支付密码和安全验证

### 2.4 通知系统
- 接收任务状态变更通知
- 接收系统公告和活动信息
- 管理通知偏好设置

### 2.5 工单系统
- 提交问题和建议
- 跟踪工单处理进度
- 与客服人员沟通

### 2.6 AI 评论生成
- 使用 AI 生成评论内容
- 自定义评论模板和参数
- 管理评论库

## 3. 使用规则

### 3.1 账户规则
- 每个用户只能注册一个账户
- 账户不得转让、出租或出售
- 账户密码应定期更换

### 3.2 任务规则
- 任务内容应符合法律法规和平台规定
- 任务发布者应确保任务描述清晰准确
- 任务参与者应按照要求完成任务

### 3.3 交易规则
- 交易应通过平台指定的支付方式进行
- 交易金额应与任务或租赁服务的实际价值相符
- 交易完成后，资金将按照平台规则进行分配

### 3.4 内容规则
- 平台上发布的内容应符合法律法规和公序良俗
- 不得发布违法违规、淫秽色情、暴力恐怖等内容
- 不得发布侵犯他人知识产权、隐私权等合法权益的内容

## 4. 收费标准

### 4.1 任务服务收费
- 平台可能对任务发布收取一定的服务费用
- 具体收费标准将在任务发布页面显示
- 费用将在任务发布时扣除

### 4.2 账号租赁收费
- 平台对账号租赁交易收取一定比例的服务费用
- 具体收费比例将在租赁交易页面显示
- 费用将在交易完成时自动扣除

### 4.3 其他服务收费
- 平台可能对其他增值服务收取费用
- 具体收费标准将在相关服务页面显示

## 5. 服务流程

### 5.1 任务发布流程
1. 登录平台账户
2. 进入任务发布页面
3. 填写任务信息和要求
4. 确认任务详情和费用
5. 提交任务

### 5.2 任务参与流程
1. 浏览任务列表
2. 选择适合的任务
3. 按照任务要求完成任务
4. 提交任务结果
5. 等待任务审核和结算

### 5.3 账号租赁流程
1. 浏览账号租赁市场
2. 选择适合的账号
3. 提交租赁申请
4. 支付租赁费用
5. 开始使用账号
6. 租赁期满后归还账号

### 5.4 提现流程
1. 登录平台账户
2. 进入钱包页面
3. 点击提现按钮
4. 填写提现金额和银行卡信息
5. 提交提现申请
6. 等待资金到账

## 6. 安全提示

### 6.1 账户安全
- 不要向他人透露账户密码和验证码
- 不要使用公共电脑或不安全的网络登录账户
- 定期检查账户活动记录

### 6.2 交易安全
- 只通过平台指定的支付方式进行交易
- 不要与其他用户私下交易
- 仔细核实交易对方的身份和信息

### 6.3 信息安全
- 不要在平台上发布个人敏感信息
- 不要点击来源不明的链接
- 定期更新密码和安全设置

## 7. 常见问题

### 7.1 账户问题
- 忘记密码：可以通过手机或邮箱找回
- 账户被冻结：联系客服了解原因并解决
- 信息修改：在个人中心页面修改账户信息

### 7.2 任务问题
- 任务审核时间：一般为1-2个工作日
- 任务失败原因：查看任务详情和失败原因
- 任务结算时间：任务审核通过后立即结算

### 7.3 交易问题
- 支付失败：检查支付方式和账户余额
- 提现到账时间：一般为1-3个工作日
- 交易纠纷：通过工单系统提交纠纷
